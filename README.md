# موقع توعوي عن السرطان في الأردن

محتوى المشروع: موقع ثابت (HTML/CSS/JS) يعرض معلومات عامة، بطاقات توعوية، وقسم لإظهار إحصاءات مأخوذة من مصادر رسمية.

ملفات مهمة:
- `index.html` — واجهة الموقع (لغة عربية، RTL).
- `css/styles.css` — أنماط CSS للتصميم والاستجابة.
- `js/app.js` — تحميل بيانات JSON وعرض البطاقات والإحصاءات.
- `data/sources.json` — ملف البيانات والمصادر (يمكن تحديثه بالأرقام الحقيقية).

كيفية التشغيل محليًا:
1. افتح المجلد `c:\majd` في محررك.
2. افتح `index.html` في المتصفح (ثنائي النقر) أو شغِّل سيرفر بسيط للملفات الثابتة إن رغبت.

الملاحظات والمرحلة التالية:
- حاليا تركت خانة الأرقام كـ "قابل للتحديث" ووضعنا روابط مصادر موثوقة (GLOBOCAN، KHCC، وزارة الصحة). أستطيع الآن استخراج الأرقام المحددة (مثل: عدد الحالات الجديدة لكل سنة، معدل الوفيات، نسب الانتشار حسب النوع) وتعبئتها في `data/sources.json` تلقائياً أو يدوياً. أخبرني إن أردت أن أملأ الأرقام الحالية من GLOBOCAN/مركز الحسين مباشرة وستتم إضافتها مع الاقتباسات.
